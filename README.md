Memory Game Challenge

A responsive memory card game built with React, Vite, and Tailwind CSS created as part of a frontend technical assessment.

## LIVE DEMO

Netlify Live Demo: https://memory-game-assessment.netlify.app/

GitHub Repository: https://github.com/AAdrianML/memory-game-assessment

## TECH STACK

* React
* Vite
* Tailwind CSS
* Recoil


## INSTALLATION

Clone the repository:

git clone https://github.com/AAdrianML/memory-game-assessment

Install dependencies:

npm install

Run the project:

npm run dev

## FEATURES

Required Features:

* Animated start screen
* Card flip animations
* Randomized cards
* Match / mismatch validation
* Sound effects
* 30 second timer
* Win / lose states
* Responsive design
* Mute / unmute functionality

Extra Features Added:

* Dark mode
* Leaderboard
* Home button to return to the main screen
* GitHub repository shortcut button
* Recoil global state management

## TECHNICAL DECISIONS

React + Vite

I chose React because the component-based structure works really well for separating screens and game logic. Vite was used for its fast development environment and lightweight setup.

Tailwind CSS

Tailwind made it easier to build a responsive interface quickly while keeping styling clean and scalable. It was especially useful for animations, responsive layouts, hover effects, and dark mode support.

Recoil

During the interview process I mentioned having experience with Recoil and atomic state management, so I wanted to include it in the project as well.

I used Recoil mainly for:

* Dark mode state
* Audio mute state

This helped keep the state centralized and avoided unnecessary prop drilling between components.
